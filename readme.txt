-------------------------
 OVERVIEW
-------------------------

The Newswriter is a simple script to publish and administrate your News.
There is no need for a database. The News a stored in a simple file.
I've coded this script because the others I found were to complicated and
not adaptable enough.

--------------------------
 REQUIREMENTS
--------------------------

- PHP 4 or 5
- No need for a Database
- FTP access

--------------------------
 INSTALLATION
--------------------------

Step 1:
Open the config.php in your texteditor and configure it.

Step 2:
Upload all files to your server except the .txt files of course.

Step 3:
Change the content.dat user permission to chmod 777
and if you like move it to a subfolder with a .htaccess-file.

Step 4:
Go to your browser to access the admin script type e.g: http://yoursite.com/admin.php
and start writing your first news ;)


------------------------------
Enjoy the Newswriter!
